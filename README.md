# PDF转长图工具

一个简单易用的浏览器扩展，可以将 PDF 文件快速转换为一张长图片。

## 功能特点

- 📄 将多页 PDF 转换为一张长图
- 🎨 支持 PNG 和 JPEG 两种格式
- ⚡ 可调节清晰度（1x - 3x）
- 💾 本地处理，保护隐私

## 安装方法

### Chrome / Edge 浏览器

1. 下载本项目所有文件到本地文件夹
2. 打开浏览器，访问扩展管理页面：
   - Chrome: 地址栏输入 `chrome://extensions/`
   - Edge: 地址栏输入 `edge://extensions/`
3. 开启右上角的「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择本项目的文件夹
6. 安装完成！

## 使用方法

1. 点击浏览器工具栏的扩展图标
2. 点击「选择 PDF 文件」上传文件
3. 调整输出格式和清晰度（可选）
   - PNG：无损格式，体积较大
   - JPEG：体积小，适合分享
4. 点击「开始转换」
5. 等待转换完成后点击「下载长图」

## 注意事项

- 页数过多或清晰度过高可能导致图片过大
- 建议单次转换不超过 50 页
- 推荐倍率设置为 2.0x，平衡清晰度和文件大小

## 技术说明

- 使用 PDF.js 进行 PDF 解析和渲染
- 使用 Canvas API 进行图片拼接
- 纯前端实现，无需服务器
